🚦 Raksha AI

Intelligent Road Safety Ecosystem


“Preventing accidents. Saving lives. Empowering citizens.”

🌍 Problem Statement

India records one of the highest numbers of road accidents globally. Major challenges include:

❌ Delayed Emergency Response

Critical delays during accidents reduce survival chances.

❌ Poor Infrastructure Monitoring

Road damages often go unnoticed and unreported.

❌ Lack of Real-Time Insights

Citizens lack live safety information and alerts.

❌ Limited Awareness

Traffic rules and penalties are not easily accessible.

💡 Our Solution

Raksha AI is a unified AI-powered platform designed to:

  • 🚨 Respond instantly during accidents
  • 🛣️ Detect and report road issues using AI
  • 📊 Visualize accident-prone zones
  • ⚠️ Predict and warn about high-risk areas

🔥 Key Features

🚨 Smart SOS System

  • One-tap emergency alert
  • Real-time location sharing
  • Nearby hospitals & emergency services
  • Ambulance ETA simulation

🛣️ AI Road Issue Detection

  • Upload road images
  • Pothole detection
  • Damaged road classification
  • Confidence score analysis

📊 Interactive Dashboard

  • Accident hotspot visualization
  • Road issue mapping

⚠️ Accident Risk Alert

  • AI-based risk prediction

  • Real-time warning alerts

⚖️ Legal Awareness Module

  • Traffic law information
  • Fine awareness UI
  • Location-based rules (future scope)

⚙️ Installation & Setup

🔹 Clone Repository

git clone https://github.com/your-username/raksha-ai.git
cd raksha-ai

🔹 Frontend Setup

cd frontend
npm install
npm start

🔹 Backend Setup

cd backend
python -m venv venv
venv\Scripts\activate

pip install fastapi uvicorn
uvicorn main:app --reload

🔹 AI Module

cd ai-models
python pothole_detection.py
